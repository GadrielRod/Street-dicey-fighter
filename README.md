# Street-dicey-fighter
Street Dicey Fighter. É um jogo via terminal que combina a estratégia do Jo-Ken-Pô (PEDRA, PAPEL, TESOURA) com gerenciamento de dados e combos únicos de personagens como Ryu e Ken. Desenvolvido em Python com POO, módulos e interface via terminal. Você pode jogar a versão mais recente de Street Dicey Fighter diretamente no seu navegador através do OnlineGDB:
https://onlinegdb.com/498j29mPc

# Street Dicey Fighter

Street Dicey Fighter is a terminal-based game that blends the strategy of Rock–Paper–Scissors (ROCK, PAPER, SCISSORS) with dice management and unique character combos inspired by fighters such as Ryu and Ken.
It is developed in Python, using Object-Oriented Programming, modular design, and a terminal-based interface.

You can play the latest version of Street Dicey Fighter directly in your browser via OnlineGDB:
https://onlinegdb.com/498j29mPc

# Latest Update 4.0

Files updated: main.py, utils.py, engine.py, and characters.py

Added new playable characters, including E. Honda and Dhalsim.

Akuma and M. Bison are now the final bosses in Tournament Mode. They can also be made playable by calling their classes in engine.py and main.py, following the existing character references.

Tournament Mode now features two difficulty levels: Normal and Hard.

There is a 50% chance for the final boss to be either Akuma or M. Bison.
