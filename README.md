# Street-dicey-fighter
Street Dicey Fighter. É um jogo via terminal que combina a estratégia do Jo-Ken-Pô (PEDRA, PAPEL, TESOURA) com gerenciamento de dados e combos únicos de personagens como Ryu e Ken. Desenvolvido em Python com POO, módulos e interface via terminal. Você pode jogar a versão mais recente de Street Dicey Fighter diretamente no seu navegador através do OnlineGDB:
https://onlinegdb.com/RTP6oM3_0R
